Quick Integration

// Server-side: Create a payment
const response = await fetch('https://api.nexuspay.in/v1/payments/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Api-Key': 'nxp_test_your_key_here'
  },
  body: JSON.stringify({
    amount: 49900,  // ₹499.00 in paise
    currency: 'INR',
    order_id: 'ORD-001',
    customer: { name: 'Customer', email: 'c@e.com' }
  })
});

For full API documentation visit your API Keys page. Use test mode for development.
